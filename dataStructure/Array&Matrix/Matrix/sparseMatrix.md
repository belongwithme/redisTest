# 稀疏矩阵 (Sparse Matrix) 详解

> 提示：此文件用于系统性梳理稀疏矩阵的概念、表示与运算；详细实现可参考后续补充的CSR/CSC/COO代码模板。

## 1. 概念与动机
- 稀疏矩阵定义与稀疏度/密度
- 为什么需要压缩：节省空间、提升运算效率
- 应用：图、推荐系统、科学计算、搜索引擎

## 2. 常见存储格式
- Triplet（三元组）
- COO（坐标格式）
- CSR（压缩行）
- CSC（压缩列）
- 对称/三角/带状矩阵的专用压缩

## 3. 基础运算
- 稀疏矩阵加法（同格式）
- 稀疏矩阵乘法（CSR×CSC）
- 稀疏矩阵与向量乘（SpMV）
- 稀疏矩阵转置

## 4. 选择指南
- 构建阶段：COO
- 行访问/SpMV：CSR
- 列访问/转置友好：CSC
- 稀疏度阈值与密集存储切换

## 5. 实战建议
- 统一索引与有序性要求
- 避免频繁重分配：预估nnz或分块构建
- 与缓存友好的分块乘法结合
